## Global Service - Profile: Modify Region

URL: https://global-profile-service.game-social.epicgames.com/profile/play_region \
Method: PUT \
Auth Required: Yes (No Perm required)

```json
{
  "namespace": "Fortnite",
  "play_region": "EUROPE"
}
```

## Parameters

`namespace`: e.g. `Fortnite` \
`play_region`:

- UNDEFINED_REGION
- NA_CENTRAL
- NA_EAST
- NA_WEST
- EUROPE
- OCEANIA
- BRAZIL
- ASIA
- MIDDLE_EAST
