# Get Code Availability

URL: https://sac.epicgames.com/api/get-code-availability \
Method: GET \
Auth Required: Yes

## Query Parameters

`code`: required SaC Code to check

---

_Example Response_

```json
{
  "data": {
    "slug": "github_lele",
    "available": true,
    "promptCRCC": false
  },
  "success": true
}
```
