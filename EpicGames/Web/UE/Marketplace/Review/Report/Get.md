## UE Marketplace - Get Review Report

URL: https://www.unrealengine.com/marketplace/api/review/:type/:topicId/report \
Method: GET \
Auth Required: Yes

## Path Parameters

`type`: The Type of the Review (reviews, questions, replies) <br/>
`topicId`: The Topic Id of the Review
